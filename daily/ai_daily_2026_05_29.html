万亿前夜

这一轮把"AI 创业公司估值上限"重新锚位 —— 不是 OpenAI 的 $852B，是 Anthropic 的 $965B。$47B 年化营收、+130% 增速预期、首个经营利润的可能性，三个数字共同把市场推到「IPO 前最后一次私募」的门口。

💰 头条 · ANTHROPIC SERIES H

Anthropic 完成 $65B H 轮 · 投后估值 $965B · 一夜超越 OpenAI

5 月 28 日，Anthropic 官宣完成 $65B 的 Series H 融资，投后估值锁在 $965B —— 距上一轮 $183B 的 Series G 仅 8 个月，估值翻 5.3 倍，一举把 OpenAI 三月份 $852B 的纪录挤下王座。本轮由 Altimeter、Dragoneer、Greenoaks、红杉资本、Capital Group、Coatue、D1 联合领投；机构方还包括 Baillie Gifford、黑石、博枫、D.E. Shaw Ventures、DST、富达；战略基础设施伙伴三星、SK 海力士、美光首次以股东身份入场；亚马逊四月份承诺的 $5B 一并并入，叠加超大规模云厂的 $15B 既有承诺。叙事的三个数字是：5 月初已确认的 $47B 年化营收（穿过 OpenAI 同期）、《华尔街日报》本周报道今年营收将翻 130%、跑出公司创办以来首个经营利润。这意味着 IPO 前的最后一次私募已经被定价为「准上市公司估值」—— 单独看是一份融资公告，叠加同日发布的 Opus 4.8 与上量在即的 Mythos 一起看，是一家 AI 公司在 12 个月内把"营收 / 安全 / 价格 / 渠道"四件事同时跑通的现场记录。

模型与基建 同步升档

融资是台前，产品与基建是台后。Opus 4.8 在同一天上线，给模型继续抬"诚实度"与"工作时长"；远在巴黎，Mistral 把 $830M 拍在 44MW 数据中心上 —— 一家做软件，一家做电；两条路同时给"AI 主权"补桩。

🛠️ 模型新版 · CLAUDE OPUS 4.8

Anthropic 发布 Opus 4.8 · Dynamic Workflows 上桌 · Mythos 数周内推

与 H 轮同日，Anthropic 上线 Claude Opus 4.8。新模型保持 Opus 4.7 的同价（$5/百万输入 token、$25/百万输出 token），但 agentic coding 从 64.3% 升到 69.2%、多学科推理工具使用从 54.7% 升到 57.9%；2.5 倍速的「Fast Mode」让生产环境单位 token 价继续往下压。两个新能力同步落地 —— Dynamic Workflows 让 Opus 在跑大型任务时自动拆出数百个并行子代理；Messages API 允许在不破坏 prompt cache 的情况下中途插入 system 指令。同时 Anthropic 确认 Mythos 将在数周内开放给全部企业客户 —— 这台「比 Opus 4.7 整整高一档」的模型已经在 Project Glasswing 内部与亚马逊、微软、苹果、Google 等约 50 家合作伙伴里跑出了 10,000 多个高危漏洞修复，下一阶段是从安全研究台搬到 Claude Code 与企业 API。

🇪🇺 欧洲算力 · MISTRAL

Mistral 再拿 $830M 债务 · 巴黎 44MW 数据中心 6 月点亮

5 月 28 日 Mistral AI 宣布再获 $830M 债务融资，由法国巴黎银行（BNP Paribas）牵头、加上法国国家投资银行 Bpifrance 等参与。资金全部投入巴黎郊区 44MW 自建数据中心（6 月正式投运），同时为 Vibe 软件栈与正在设计的自研芯片提供启动现金。把这笔放回时间轴看 —— 3 月 Mistral 拿过一笔 €722M、6 月节点是把巴黎集群通电，2027 年底前瞄准的是欧洲范围内 200MW 自有算力。Mistral 的转向已经清晰：从纯模型厂转成「全栈 + 主权」基建商，把对话上层留给 Le Chat，把训练 / 推理底座抓在自己手里。这一周，欧洲 AI 自留地的故事，靠的不是融资数字，是电力数字。

📄 arXiv · 2605.05440

多代理系统的「授权传播」· 把身份治理写进基础设施层

5 月底 arXiv 2605.05440《Authorization Propagation in Multi-Agent AI Systems: Identity Governance as Infrastructure》提出：当代理之间互相调用、互相代办时，原本静态的「人 → 系统」权限模型会塌陷，必须把「身份治理」当作基础设施层来做。论文把问题拆成三块 —— 传递式委托（A 委托 B 之后 B 又委托 C，授权链如何不失真）、聚合推断（多个代理同时拉一份数据，原始 token 权限如何回算）、时间有效性（代理任务跨小时甚至跨天时，旧令牌如何撤回与续约）。读法上 —— 把它跟本周 KPMG「276,000 人桌面跑 Claude」与 Anthropic Managed Agents 的多代理编排放在一起，这篇就是给企业级代理上量补一道法理：当代理跑得久、跑得多、跑进客户数据，治理不能再事后追加。
