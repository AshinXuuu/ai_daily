📄 ARXIV · MEMREREAD

MemReread · Agent 长上下文「主动重读记忆」机制

arXiv 2605.10268（5 月 11 日挂出）：来自 Baibei Ji 等团队的一份 cs.CL 工作。核心问题 —— 现有 RAG / Agent Memory 都是"检索一次就用"，但长链路推理里关键记忆经常被推理早期的噪声覆盖。MemReread 提出"记忆引导的重读"：让 Agent 在推理中识别记忆和当前推理状态的张力，触发一次定向回看，再做下一步动作。这一篇和上周提到的《From Storage to Experience》形成方法学链 —— Agent Memory 在 2026 上半年已经从"存什么"演进到"什么时候回头看什么"。模型上下文窗口 2M tokens 再大也没用，「读得到 ≠ 用得到」。这是研究界给 Gemini 4 / Claude / ChatGPT 持久化 Agent 路线提前预埋的下半场答案。
