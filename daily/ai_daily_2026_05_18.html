⚡ 头条 · OPENAI DEPLOYCO

OpenAI 联手 19 家投行咨询砸 $40 亿建 DeployCo · 同步收 Tomoro 把 150 名工程师塞进去

5 月 11 日 OpenAI 官宣 OpenAI Deployment Company（DeployCo） — 一只由 OpenAI 主导、19 家全球资管、咨询、集成商联合发起的「企业 AI 落地公司」，初始投入 $40 亿。TPG 领投，Advent / Bain Capital / Brookfield 联合创始合伙人；B Capital、BBVA、Emergence、Goanna、Goldman Sachs、SoftBank、Warburg Pincus、WCAS 跟投。咨询端拉了 Bain & Company、Capgemini、McKinsey & Company 入伙。同日 OpenAI 收购 AI 咨询公司 Tomoro，把约 150 名 Forward Deployed Engineer 直接装进 DeployCo 第一波编制。打法很明确：让 OpenAI 的工程师常驻客户内部，把模型接到业务数据上、重塑工作流，按结果而不是按 token 计费。这是 OpenAI 第一次明确承认 — 仅卖 API 已经撑不起 $1 万亿估值。Salesforce、Accenture、IBM 也是几十年前从「卖软件 → 卖咨询 → 卖解决方案」走过这条路 — AI 巨头正在压缩走完同样的转型路径。

🧬 SELF-IMPROVING AI · RECURSIVE

Recursive Superintelligence 出隐身 · $6.5 亿、估值 $46.5 亿，要做会改自己代码的 AI

5 月 13 日伦敦初创 Recursive Superintelligence 出隐身：单轮拿 $6.5 亿、估值 $46.5 亿，GV（Google Ventures）和 Greycroft 联合领投，Nvidia、AMD、Conviction、Index Ventures 跟投。创始团队不到 30 人，CEO 是前 Salesforce 首席科学家 Richard Socher，CTO Tim Rocktäschel 是 UCL 教授 + 前 DeepMind 研究员，其余成员来自 Meta 和 OpenAI 早期。它要解的问题极端浪漫 — 训练一个能分析自身性能、识别自己的弱点、然后改写自己代码持续优化的 AI 系统。术语叫「Level 1 自治训练」，公开发布定 2026 年中。这是 OpenAI、Anthropic 主流路线之外，资本第一次大额下注「让 AI 自己改自己」 — 也是 LeCun 的 AMI Labs（$10 亿 seed 押 world models）之后，欧洲 / 跨大西洋阵营的第二次真正意义上的「另一条路」融资。AI 路线之争开始有真金白银的分叉了。

英国 AI 安全研究所发布 Claude Mythos Preview 的官方评测 — 第一次把一只「能自己跑完 32 步企业网络入侵」的前沿模型，摊给监管机构和企业 CISO 看。AI 安全从论文话题，变成了董事会议题。

🛡️ AI 安全 · CLAUDE MYTHOS / AISI

英国 AISI 实测 Claude Mythos Preview · 73% 通关专家级 CTF · 3/10 跑通 32 步企业网络全链路

英国 AI Security Institute 在 5 月中旬发布对 Anthropic Claude Mythos Preview 的官方网络安全能力评测 — 这是 Anthropic 在 Claude Opus 之上的非公开前沿模型，仅向受邀的 Project Glasswing 联盟成员（AWS、Apple、Microsoft、Google、CrowdStrike、Palo Alto Networks 等）开放。AISI 测出：Mythos 在专家级 CTF 上 73% 一次通关；在内部命名为「The Last Ones」的 32 步企业网络模拟攻击（侦察 → 横向移动 → 域控接管，正常需 20 小时红队工时）里，Mythos 10 次尝试中 3 次完整跑通、平均完成 22 / 32 步。同步 Anthropic 披露 Mythos 已在主流操作系统和浏览器里识别出数千个 0-day，包括 OpenBSD 上一个 27 年的老 bug、FreeBSD NFS 一个组合 6 个 RPC 请求的远程代码执行。这是 frontier AI 安全评估第一次明确给出「能 / 不能」的二元答案 — 也意味着监管端从「模型是不是好用」彻底转向「模型是不是危险」。模型公司既是防御侧的 vendor，也是攻击侧的潜在源头 — 双面刃定价权回到了能造它的人手里。

📄 综述论文 · ARXIV 2605.06716

《From Storage to Experience》· Agent Memory 进入「演化」方法学

5 月新挂出的 arXiv 2605.06716 是一份系统化综述：从最早的纯向量记忆 → RAG 检索 → 工作记忆 / 情景记忆分层 → 强化学习训出来的「记忆动作」（store / retrieve / update / summarize / discard），把 LLM Agent Memory 重组成一条演化轴。AgeMem 类工作（三阶段 RL + step-wise GRPO）被作为新一代代表 — 模型不再只是「记得多」，而是「会主动整理记忆」。当 OpenAI、Anthropic、Google 都在拼上下文窗口时，学术界把焦点拨回到「记忆即操作」这条更长远的路。

💎 GITHUB TRENDING · KARPATHY / SKILLS

Karpathy 的 CLAUDE.md · 一个人手写的「Claude 用法说明」登顶 GitHub Trending

5 月第二周 GitHub Trending AI 类目里登顶的不是新模型、不是新框架 — 而是 Andrej Karpathy（前特斯拉 AI 总监 / OpenAI 创始成员）一个人手维护的 CLAUDE.md 仓库：一份基于他自己观察到的 LLM 编程常见坑而写的「Claude Code 使用规范」，从 prompt 排版、到调用约束、到失败重试逻辑，全是经验法则。这是 Karpathy 风格典型 — 没有正式模型、没有产品发布，只把自己用 Claude 写代码的真实笔记开放出来。同周 Trending 还有 ruflo、agent-skills、DeepSeek-TUI、Anthropic 的 financial-services 仓库 — 共同信号：开发者社区已经从「炫模型」全面转向「炫 prompt 工程 / agent 工程」。模型是公共品，会用才是私产。
